# Prompt Wars Assets

This directory contains app assets including images, fonts, and other media.

## Directory Structure

- `images/` - App icons, splash screens, and UI images
- `fonts/` - Custom fonts (if any)
- `sounds/` - Music stings and sound effects (Phase 3+)

## Required Assets

For Phase 0, placeholder images are used. Production assets will be added later.

### Icons
- `icon.png` (1024x1024) - App icon
- `adaptive-icon.png` (1024x1024) - Android adaptive icon
- `favicon.png` (48x48) - Web favicon

### Splash Screen
- `splash-screen.png` (1284x2778) - Launch screen

## Notes

- All images should be optimized for mobile
- Use SVG for scalable UI elements when possible
- Sound assets will be added in Phase 3 (AI video integration)
